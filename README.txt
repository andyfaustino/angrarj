# Site da excursão - Angra dos Reis

Estrutura para publicar no GitHub Pages:

- index.html
- imagens/foto-01.webp
- imagens/foto-02.jpg
- imagens/foto-03.webp
- imagens/foto-04.jpg
- imagens/foto-05.webp
- imagens/foto-06.jpg
- imagens/foto-07.webp

Como publicar:
1. Envie o conteúdo deste ZIP para o repositório do GitHub.
2. Mantenha o index.html na raiz.
3. Mantenha a pasta imagens ao lado do index.html.
4. Ative GitHub Pages em Settings > Pages > Deploy from a branch > main > /root.
